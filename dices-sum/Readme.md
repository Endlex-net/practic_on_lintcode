# dices-sum 骰子求和 
## 描述
扔 n 个骰子，向上面的数字之和为 S。给定 Given n，请列出所有可能的 S 值及其相应的概率。
### 注意事项
You do not care about the accuracy of the result, we will help you to output results.
## 样例 
给定 n = 1，返回 [ [1, 0.17], [2, 0.17], [3, 0.17], [4, 0.17], [5, 0.17], [6, 0.17]]。
